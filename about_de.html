Über das Projekt

Dieses Projekt wurde im Rahmen eines neuen Ansatzes in der Softwareentwicklung erstellt, bei dem KI-Assistenten wie Copilot und ChatGPT bei der Code-Implementierung helfen. Design, Funktionalität und Anforderungen wurden von Rikard Svenningsen festgelegt, während KI-Assistenten beim Programmieren halfen.

Diese Version ist die neue Ausgabe der dänischen Shopping-App, daher wird die dänische Version nicht mehr gepflegt. Diese Version ist Loading....

Die Methode ermöglicht es Nicht-Programmierern, Software zu entwickeln, indem sie sich auf Design und Struktur konzentrieren, während KI den Code generiert und anpasst. Sie zeigt, dass man zur Softwareentwicklung beitragen kann, ohne selbst Code zu schreiben – einfach durch das Definieren von Anforderungen und das Testen von Lösungen.
