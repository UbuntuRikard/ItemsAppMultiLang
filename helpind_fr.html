Application de Courses

Avec cette application, vous pouvez créer des listes de courses, des rappels et des inventaires.

Pour une liste de courses :
Créez un magasin et ajoutez des catégories via le panneau d’administration (plus d’infos dans “Administration”).

Les catégories vous aident à naviguer efficacement dans un magasin sans revenir sur vos pas.

Pour commencer, allez dans “Administration”.

Pour organiser votre parcours, mettez un numéro devant le nom de la catégorie. La liste sera triée par ordre :

Catégories :

01 Pain
01 Légumes
02 Viande fraîche (non surgelée)

Dans un magasin de bricolage :

Catégories :

01 Électricité
02 Plomberie
03 Outils

Vous pouvez aussi suivre vos réserves à la maison : congélateur, garde-manger, etc.
Créez un magasin nommé “Inventaire Maison”.

Catégories :

Congélateur
Garde-manger
Buanderie

Appuyez sur la quantité pour modifier un article. Il sera déplacé en haut. Modifiez la quantité, le nom ou la catégorie. Puis cliquez sur “Mettre à jour”.

Pour ajouter un article, entrez une quantité, un nom, une catégorie. Le prix ne peut pas être saisi maintenant.

Une quantité de 0 transforme la liste en rappel. Paramètres de visibilité :

  • Afficher tous les articles
  • Masquer les articles avec 0
  • Afficher uniquement ceux avec 0

Ajouter une quantité à un article “0” le déplace vers la liste normale.

Vous pouvez entrer un prix, qui sera formaté automatiquement en “.00”.

Un article avec prix et quantité peut être marqué comme acheté. Il va en bas, le total se met à jour. Les prix négatifs (comme consignes) sont aussi gérés.

Le suivi des prix est facultatif mais utile pour le budget. Bien utilisé, l’application donne une estimation fiable.

En affichant une catégorie, seuls ses articles comptent pour le total.
