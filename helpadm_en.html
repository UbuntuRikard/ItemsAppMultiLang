Administration

This is where you create and delete store lists or general lists. Lists can represent more than just a store. Once you have created a store, you can select it and then create categories. Categories serve multiple purposes; they can indicate the precise location of an item in, for example, a hardware store, or any other retail shop with location-specific item placements. You can also use them to determine the order in which you collect items while shopping, by adding a number before the category nameâ€”for example, "01 Vegetables," where the category is sorted numerically first, then alphabetically. Within this administration panel, you can also rename categories. In this version, you cannot edit store or list names.

Available Administration Features:

  • Add Store
    Adds a store, or simply a list as you prefer
  • Delete Store
    Deletes the selected store, along with items and categories
  • Export
    Exports the entire database, including quantities, item text, prices, categories, and stores
  • Import
    Opposite of export, applies to all data

Select Store, which must be done at the top, then you can:

  • Add Category
    Categories help locate items or indicate their placement
  • Delete Category
    Deletes the category from the list, along with the items within it
  • Edit Category
    Renames the category
