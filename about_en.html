About the Project

This project was developed as part of a new approach to software development, where AI assistants such as Copilot and ChatGPT help with code implementation. The design, functionality, and requirements were specified by Rikard Svenningsen, while AI assistants helped with the coding.

This version is the new edition of the Danish shopping app, so the Danish version is no longer maintained. This version is Loading....

The method allows non-programmers to develop software by focusing on design and structure while AI generates and adjusts the code. It demonstrates that one can contribute to software development without writing code themselves – simply by defining requirements and testing solutions.
